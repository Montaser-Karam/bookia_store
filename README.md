## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- Dart SDK
- Android Studio or VS Code

---

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/bookia_store.git
```

Go to the project folder:

```bash
cd bookia_store
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```
